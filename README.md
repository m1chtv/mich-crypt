# mich·crypt

A lightweight **encryption & hashing toolkit** for developers — browser-only, local processing.

## Features
- **Encrypt / Decrypt** text
- Multiple ciphers & modes
- Hash generation (MD5, SHA-1, SHA-256, etc.)
- Clean UI with instant output
- Copy results with one click

## Tech Stack
- HTML5  
- CSS3  
- Vanilla JavaScript  

## How to Use
1. Open `index.html` in a modern browser.  
2. Enter your **plaintext** or **ciphertext**.  
3. Choose an **algorithm** and mode.  
4. Click **Run** to encrypt/decrypt or hash.  
5. Copy output for your workflow.

## Notes
- 100% client-side; no backend, no tracking  
- Useful for testing, dev tasks, and quick cryptographic ops  
- Not for sensitive production encryption without review
